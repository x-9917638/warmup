Cli tool done as warmup for py-resolution week 2

# Usage:

```
usage: warmup [-h] [--email EMAIL] [-d <ID>] [-l] [contact]

positional arguments:
  contact            Contact to add

options:
  -h, --help         show this help message and exit
  --email EMAIL      Optional contact email
  -d, --delete <ID>  Delete a contact by ID
  -l, --list         List all tasks
```
